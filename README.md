## Introduction
우리는 인터넷 상에서만 존재하고 현실에서는 큰 영향을 미치치 못하는 무언가를 '데이터 쪼가리'라고 부르곤 합니다.

데이터 조각(piece of data)는 이를 모아 여러분의 일상에 도움이 되는 하나의 조각을 만들고자하는 대학생 프로그래머 팀입니다.
